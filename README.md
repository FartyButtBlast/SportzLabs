# SportzLabs Corporate Site

Static corporate landing page for `www.sportzlabs.com`.

## Deploy

Create a separate GitHub repository for this folder, for example `SportzLabs`.

In GitHub Pages, set the custom domain to:

```text
www.sportzlabs.com
```

In GoDaddy DNS:

- `www` should be a `CNAME` to `FartyButtBlast.github.io`
- `@` should use GitHub Pages `A` records or GoDaddy forwarding to `https://www.sportzlabs.com`

The AFL Performance app should stay on:

```text
https://aflperformance.sportzlabs.com/
```

Public policy pages:

- `https://www.sportzlabs.com/privacy.html`
- `https://www.sportzlabs.com/eula.html`

Public app pages:

- `https://www.sportzlabs.com/apps/afl-performance-analytics/`

Google Analytics is installed with measurement ID:

```text
G-PYHWJMJ0R7
```

SEO files:

- `robots.txt`
- `sitemap.xml`
- canonical URLs, Open Graph tags, Twitter card tags, and schema.org structured data
